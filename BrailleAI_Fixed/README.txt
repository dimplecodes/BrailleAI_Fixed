# BrailleAI Fixed

## Step 1
Copy your best.pt file into:

backend/model/best.pt

## Step 2
Install Python packages

pip install -r requirements.txt

## Step 3
Install Node packages

cd backend
npm install express multer cors node-fetch@2 form-data

## Step 4
Run Flask

python flask_api.py

## Step 5
Run Node

cd backend
node server.js

## Step 6
Open browser

http://localhost:5000
